# Travel Agency Operations Analysis (Excel End-to-End Project)
## Project Overview
This project explores travel agency performance using operational data from August 2024 to July 2025. The goal was to uncover trends in bookings, revenue, customer satisfaction, and regional demand to support smarter business decisions.

From raw data chaos to a structured, interactive dashboard, this project demonstrates the full data analysis lifecycle using Microsoft Excel.
## Key Objectives
- Identify booking trends across months
- Evaluate agent performance based on revenue
- Understand customer preferences by trip type
- Highlight high-demand regions
- Enable dynamic filtering using slicers
## Tools & Skills Used
- Microsoft Excel
- Data Cleaning & Transformation
- Pivot Tables
- Data Visualization
- Dashboard Design
- Analytical Thinking
## Data Cleaning & Preparation
- Handled missing and inconsistent values
- Standardized date formats for time-based analysis
- Cleaned categorical fields (trip types, regions, agent names)
- Created calculated fields for deeper insights
